Real-time weather data and forecasts.

## Planned Features

- Current weather conditions
- 5-7 day forecasts
- Hourly forecasts
- Weather alerts and warnings
- Air quality index
- UV index
- Historical weather data
- Multi-location support

## Planned API Support

- OpenWeatherMap API (Free tier available)
- WeatherAPI.com (Free tier available)
- National Weather Service (US, free)

## Expected Tools

- `get_current_weather` - Current conditions for a location
- `get_forecast` - Multi-day forecast
- `get_hourly_forecast` - Hour-by-hour forecast
- `get_weather_alerts` - Active weather alerts
- `get_air_quality` - Air quality index and pollutants
- `search_locations` - Find location coordinates
