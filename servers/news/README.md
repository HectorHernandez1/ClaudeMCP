Real-time news aggregation from multiple sources.

## Planned Features

- Top headlines by category
- Search articles by keyword
- Filter by source, language, country
- RSS feed aggregation
- Article summaries
- Source credibility ratings
- Trending topics
- Historical article search

## Planned API Support

- NewsAPI.org (Free tier: 100 requests/day)
- RSS feed parsers (Free, no API key needed)
- Google News RSS (Free)

## Expected Tools

- `get_top_headlines` - Latest top stories
- `search_news` - Search articles by keyword
- `get_news_by_category` - Filter by business, tech, sports, etc.
- `get_news_by_source` - Articles from specific sources
- `get_trending_topics` - What's trending now
- `search_historical` - Find older articles
